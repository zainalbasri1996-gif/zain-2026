# solar-landing-store

متجر إلكتروني ومخطط تصميم منظومات شمسية بمواجهة احتياجات المستخدمين

هذا المشروع عبارة عن صفحة landing-page تفاعلية تجمع متجرًا إلكترونيًا لبيع ألواح شمسية وبطاريات وانفرترات وملحقات مع تصفح حسب الفئة والسعر والتقييم. يتضمن التطبيق سلة تسوق، وخيار دفع آمن، ونموذج تسجيل دخول عبر OTP أو صفحة تسجيل دخول باسم وكلمة مرور. كما يقدم منشئ تصميم منظومات شمسية يسمح بتحديد النوع والمكونات وتوليد تكلفة قابلة للحفظ والتصدير، إضافة إلى لوحة إدارة لإدارة المنتجات والعروض والطلبات والتقارير.
